# VelpTEC Prompt Engineering – Teilprüfung 2

## Overview

This repository contains the prompt artifacts created for Prompt Engineering Teilprüfung 2.

The task focuses on:

- selecting suitable prompting techniques for different scenarios
- designing a combined prompt for market analysis
- creating an iterative storytelling prompt
- documenting prompt design in a reproducible way
